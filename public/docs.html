Emerald Language Docs

A tiny, interpreted language that gets out of your way.

1. Getting Started

1.1 Installation

Download the latest release, add it to your PATH, and you're done. That's it.

Once installed, run any Emerald files with:

emerald fileName.emld

1.2 Editor Support

Search for Emerald Language in the VS Code Extensions panel to install the syntax highlighting extension for .emld files.

1.3 Hello World

print "Hello, World!"

2. Comments

Comments start with ! and run to the end of the line.

! This is a comment
print "hello" ! inline comment

3. Variables

All variables are global. Declaration uses the var. prefix. Reassignment uses a bare . prefix (no var.).

var.name "Hanno"
var.age 67
var.isCool False
var.nothing Null

.name "Bob"       ! reassignment
.age += 1         ! compound assignment
.age -= 5
.age *= 2
.age /= 3

4.1 Type Checking

Use num, str, or bool as values with the = operator to check a variable's type. Combine with not to negate.

if x = num {
  print "x is a number"
}
if s = str {
  print "s is a string"
}
if b = bool {
  print "b is a boolean"
}
if x = not str {
  print "x is not a string"
}

4.2 String Interpolation

Use $"..." to embed expressions inside strings. Non-string values must be explicitly converted with str() - no automatic coercion.

var.name "World"
print $"Hello, {name}!"     ! Hello, World!

var.score 42
print $"Score: {str(score)}"  ! Score: 42

Note: equality is a single =, not ==.

var.x 10
var.y 3
print x + y    ! 13
print x - y    ! 7
print x * y    ! 30
print x / y    ! 3.33333
print x > 5    ! True

! not negates booleans and flips equality
if not done { ... }
if x = not 5 { ... }  ! x != 5

! or expands equality to membership
if x = 1 or 2 or 3 { ... }  ! x is 1, 2, or 3

! and combines conditions
if x > 0 and y > 0 { ... }

6. Print

The print statement evaluates an expression and outputs it to stdout with a trailing newline.

print "Hello"
print 123
print myVar
print $"The value is {str(x)}"

7. Conditionals

if, elif, and else for branching. Conditions can use not, or, and and.

if x > 5 {
  print "big"
}
elif x > 0 {
  print "positive"
}
else {
  print "small or zero"
}

if not flag {
  print "flag is falsy"
}

if x > 0 and x < 10 {
  print "between 1 and 9"
}

8. Loops

Emerald supports while loops and for...in loops.

8.1 While Loops

Repeat a block while a condition is truthy. Supports not, or, and and in conditions.

var.x 5
while x > 0 {
  print x
  .x -= 1
}

8.2 Numeric Range

Use range:N to loop from 1 to N inclusive. Use range:(start, end) for a custom range.

for i in range:5 {
  print i
}
! prints 1 2 3 4 5

for i in range:(0, 3) {
  print i
}
! prints 0 1 2 3

8.2 List Iteration

var.fruits ("apple", "banana", "cherry")

for fruit in fruits {
  print fruit
}

9. Functions

Functions are defined with fn. and called with run. Functions have no parameters and no return value.

fn.greet {
  print "Hello!"
}

fn.sayName {
  print name
}

run greet
run sayName

10. Lists

Lists are parenthesised, comma-separated values. They are 1-indexed.

! Create a list
var.items (10, 20, 30)

! Access by index (1-based)
print items:1  ! 10
print items:3  ! 30

! Slice (start, length)
print items:slice(2, 2)  ! (20, 30)

! List length
print items:len  ! 3

! Add an element
add items 40

! Empty list
var.empty ()

11. String Methods

Strings support method calls via the : operator.

var.text "Hello, World!"

! Slice (start, length)
print text:slice(1, 5)  ! Hello

! Case conversion
print text:upper            ! HELLO, WORLD!
print text:lower            ! hello, world!

! Trim whitespace
print "  hi  ":trim          ! hi

! Split into list / Join list into string
var.parts text:split(", ")
print parts                    ! Hello, World!
print parts:join(" - ")      ! Hello - World!

! Replace substring
print text:replace("World", "Emerald")

! Check contains
print text:contains("World")  ! True

! Length
print text:len                ! 13

! Chaining
print text:lower:replace("hello", "hi"):upper  ! HI, WORLD!

12. Input

Read a line from stdin with an optional prompt. Returns a string.

var.name input "Enter your name: "
print $"Hello, {name}!"

13. Built-in Functions

FunctionDescriptionExample
str(x) Convert any value to its string representation str(42) → "42"
num(x) Convert a value to a number. Works on strings, booleans, Null. num("3.14") → 3.14
bool(x) Convert a value to a boolean. Falsy values: 0, "", Null, False. bool(1) → True
