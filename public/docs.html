Emerald Language Docs

A tiny, interpreted language that gets out of your way.

1. Getting Started

1.1 Installation

Download the latest release, add it to your PATH, and you're done. That's it.

Once installed, run any Emerald files with:

emerald fileName.emld

1.2 Hello World

print "Hello, World!"

2. Comments

Comments start with ! and run to the end of the line.

! This is a comment
print "hello" ! inline comment

3. Variables

All variables are global. Declaration uses the var. prefix.

var.name "Hanno"
var.age 67
var.isCool False
var.nothing Null

4. Data Types

TypeExamplesNotes
number42, 3.14Stored as float64, printed as integer when whole
string"hello"Double-quoted. Supports \n, \t, \\, \"
booleanTrue, FalseCapital first letter
NullNullAbsent value (capital N)
list(1, 2, 3)Parenthesised, comma-separated. 1-indexed.

4.1 String Interpolation

Use $"..." to embed expressions inside strings. Non-string values must be explicitly converted with str() - no automatic coercion.

var.name "World"
print $"Hello, {name}!"     ! Hello, World!

var.score 42
print $"Score: {str(score)}"  ! Score: 42

var.x 10

print x + y    ! 13
print x - y    ! 7
print x * y    ! 30
print x / y    ! 3.33333
print x > 5    ! True

6. Print

The print statement evaluates an expression and outputs it to stdout with a trailing newline.

print "Hello"
print 123
print myVar
print $"The value is {str(x)}"

7. Conditionals

if, elif, and else for branching. Conditions must evaluate to a boolean.

if x > 5 {
  print "big"
}
elif x > 0 {
  print "positive"
}
else {
  print "small or zero"
}

8. Loops

The for...in loop iterates over a range or a list variable.

8.1 Numeric Range

Use range:N to loop from 1 to N inclusive. Use range:(start, end) for a custom range.

for i in range:5 {
  print i
}
! prints 1 2 3 4 5

for i in range:(0, 3) {
  print i
}
! prints 0 1 2 3

8.2 List Iteration

var.fruits ("apple", "banana", "cherry")

for fruit in fruits {
  print fruit
}

9. Functions

Functions are defined with fn. and called with run.. Functions have no parameters and no return value.

fn.greet {
  print "Hello!"
}

fn.sayName {
  print name
}

run.greet
run.sayName

10. Lists

Lists are parenthesised, comma-separated values. They are 1-indexed.

! Create a list
var.items (10, 20, 30)

! Access by index (1-based)
print items(1)  ! 10
print items(3)  ! 30

! Add an element
add.items 40

! Empty list
var.empty ()

11. Input

Read a line from stdin with an optional prompt. Returns a string.

var.name input "Enter your name: "
print $"Hello, {name}!"

12. Built-in Functions

FunctionDescriptionExample
str(x) Convert any value to its string representation str(42) → "42"
num(x) Convert a value to a number. Works on strings, booleans, Null. num("3.14") → 3.14

13. CLI Usage

emerald <file.emld>     Run a script
emerald update          Self-update to the latest version
emerald version         Print version number
