Emerald Examples

A collection of Emerald programs to get you started.

Hello World

The classic starter program.

print "Hello, World!"

Variables & Reassignment

Declare, reassign, and use compound assignment.

var.x 10
var.name "Emerald"

.x += 5
.name "Ruby"

print x     ! 15
print name  ! Ruby

If / Elif / Else

Branching with not, and, and or.

var.score 85

if score >= 90 {
  print "A"
}
elif score >= 80 {
  print "B"
}
else {
  print "C"
}

! with not, and, or
if score > 0 and score < 100 {
  print "valid"
}
if score = not num {
  print "not a number"
}

While Loop

Count down with a while loop and compound assignment.

var.x 5
while x > 0 {
  print x
  .x -= 1
}
! prints 5 4 3 2 1

For Loop with Range

Iterate over a numeric range.

for i in range:5 {
  print i
}
! prints 1 2 3 4 5

for i in range:(0, 3) {
  print i
}
! prints 0 1 2 3

Break & Continue

Control loop execution with break and continue.

! break: stop at 5
var.x 1
while x < 10 {
  if x = 5 { break }
  print x
  .x += 1
}
! prints 1 2 3 4

! continue: skip 3
var.y 0
while y < 5 {
  .y += 1
  if y = 3 { continue }
  print y
}
! prints 1 2 4 5

List Iteration

Loop over a list with for...in.

var.fruits ("apple", "banana", "cherry")
for fruit in fruits {
  print fruit
}

Lists

Create, access, slice, and modify lists.

! create
var.items (10, 20, 30, 40)

! access (1-based)
print items:1         ! 10

! slice (start, length)
print items:slice(2, 3)  ! (20, 30, 40)

! length
print items:len          ! 4

! add
add items 50

! iterate
for i in items {
  print i
}

Functions

Define and call functions. All variables are global.

fn.greet {
  print "Hello!"
}

fn.sayName {
  print name
}

var.name "Bob"
run greet   ! Hello!
run sayName ! Bob

Input

Read a line from stdin with a prompt.

var.name input "Enter your name: "
print $"Hello, {name}!"

Type Checking

Use type keywords with = to check variable types.

var.a 42
var.b "hello"
var.c True

if a = num  { print "a is a number" }
if b = str  { print "b is a string" }
if c = bool { print "c is a bool" }
if a = not str { print "a is not a string" }

Membership with or

Check if a value matches any of several options.

var.x 3
if x = 1 or 2 or 3 {
  print "x is 1, 2, or 3"
}

! combine with not
if x = not 0 or not 1 {
  print "x is neither 0 nor 1"
}

String Methods

Use the : operator to slice, transform, and inspect strings.

var.text "Hello, World!"

! slice (start, length)
print text:slice(1, 5)          ! Hello

! case conversion
print text:upper                  ! HELLO, WORLD!
print text:lower                  ! hello, world!

! split / join
var.parts text:split(", ")
print parts:join(" - ")          ! Hello - World!

! replace and check
print text:replace("World", "Emerald")
print text:contains("World")      ! True

! length and chaining
print text:len                      ! 13
print text:lower:replace("hello", "hi"):upper

Built-in Functions

Convert between types with str(), num(), bool().

var.n num("42")
var.s str(123)
var.b bool(1)

print n             ! 42
print $"Hello {s}"  ! Hello 123
print b             ! True
