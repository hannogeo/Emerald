Why Emerald?

Zero Setup

Download the exe, add it to PATH, and you're done. No dependencies, no runtime, no npm install.

Readable Syntax

Clean, minimal syntax inspired by a blend of scripting languages. What you see is what you get.

Built-in Updater

Run emerald update and it fetches the latest version automatically. Stay current without effort.
